Imgenes y multimedia.

Al ser una sintaxis para crear una pgina web, R Markdown nos permite incluir multimedia, tal como imgenes, video u otras pginas web.

Imgenes.

Por ejemplo, queremos colocar la siguiente imagen en nuestro documento:

Entonces, para esto, necesitamos escribir en nuestro documento el siguiente cdigo:

Siendo la sintaxis general la siguiente:

![texto_alternativo](ubicacion_de_la_imagen){width=width height=height}

Instrucciones para incluir una imgen en R Markdown:

  • Primero escribimos ![]().

  • Despus escribimos, dentro de los corchetes, el texto alternativo. Este es opcional y solo entra en accin cuando no se puede cargar la imagen correctamente.

  • Despus escribimos, dentro de los parntesis, la ubicacin del archivo (ya sea una url o una ubicacin dentro de algun folder local).

Introduciendo imgenes y videos con HTML y CSS.

Otra forma de incluir multimedia con HTML y CSS es mediante los tags de HTML. En las siguientes lineas de cdigo, vamos a mostrar un par de ejemplos sobre como podramos incluir videos o imgenes dentro de nuestros documentos.

Para incluir un video de Youtube, hay que pegar el siguiente cdigo:

<iframe width="600" height = "420"
src="https://www.youtube.com/embed/qKf2EwInKbA">
</iframe>

Para incluir un video de Youtube centrado

<div>
<p style = 'text-align:center;'>
<iframe width="600" height = "420"
src="https://www.youtube.com/embed/tgbNymZ7vqY">
</iframe>
</div>
</p>

Para incluir una imgen: 

<img src="https://pbs.twimg.com/media/EiAuAS8VoAA1-4T?format=jpg&name=large" alt="JuveR" width="300px">

Para incluir y centrar una imgen: 

<div>
<p style = 'text-align:center;'>
<img src="https://pbs.twimg.com/media/EiAAHfoU4AADis4?format=jpg&name=small" alt="JuveYell" width="300px">
</p>
</div>
